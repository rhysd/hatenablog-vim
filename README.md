# hatenablog-vim

HTML5 + inline SVG omnicomplete funtion, indent and syntax for Vim.
Based on the default htmlcomplete.vim.

## Installation

- Install with [NeoBundle](https://github.com/Shougo/neobundle.vim)

Add to vimrc:

    NeoBundle "toyamarinyon/hatenablog-vim"

And install it:

    :source ~/.vimrc
    :NeoBundleInstall
